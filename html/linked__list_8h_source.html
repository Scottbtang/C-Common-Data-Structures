127 /**
128  * @brief Insert an item to the designated index of the list.
129  *
130  * This function inserts an item to the designated index of the list and shifts
131  * the trailing items one position to the tail. The index can be positive or
132  * negative, but its absolute value should be smaller than or equal to the list
133  * size.
134  * The operation supports both forward and backward indexing. For the former one,
135  * the list is traversed from the head to the tail. And for the later one, the
136  * direction is reversed.
137  * To illustrate the behavior, let N denote the list size. Thus for forward indexing
138  * , index (0) and index (N-1) point to the head and tail. And for backward indexing,
139  * index (-1) and index (-N) point to the list tail and head.
140  * But no matter which indexing method is applied, the list always grows from the
141  * head to the tail.
142  *
143  * @param self The pointer to the LinkedList structure
144  * @param item The designated item
145  * @param iIdx The designated index
146  *
147  * @retval SUCC
148  * @retval ERR_NOINIT Uninitialized container
149  * @retval ERR_NOMEM Insufficient memory for list extension
150  * @retval ERR_IDX Out of range indexing
151  *
152  * @note The absolute value of the index should be smaller than or equal to the
153  * list size.
154  */
155 int32_t ListInsert(LinkedList *self, Item item, int32_t iIdx);
